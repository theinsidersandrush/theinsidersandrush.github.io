A year (almost) without Twitter

Published on Sunday, 18 May 2025.

Did I want to return? No.

Hello, I'm The Insider Sandrush. It's been a year since I've stop Tweeting on Twitter entirely, and since that, everything seems to go opposite.

Where do you moved, then?

I mostly on Bluesky. It's a decentralised* social media platform operating with their own AT Protocol. Begin as a side Twitter project in 2019 before being independent in 2021 from Twitter, and in May 2025 with Jack Dorsey leaving the Social board.

I'm also on Mastodon, via Pawoo server. It's a decentralised social media platform based on ActivityPub protocol. The server which I'm in, is run by Sujitech.

I started my journey on Bluesky in 3 November 2023, thanks to Scarristo for giving me one invite key on Twitter, back when Bluesky was still invites-only.

I joined Mastodon in November 2022 during Elon's first month of Twitter ownership, reinstated conspiracy theorists and other banned accounts on Twitter, firing almost all employees, banning the owner of the account ElonJet (flight tracker for Elon Musk's private jet), journalists who speaks out against him, users who put their Mastodon handle in their profile, and many more. If you want to know more about that, read this Wikipedia article.

Mastodon's server code are also being used to create alt-tech platforms, like Gab, and Truth Social (aka Donald Trump's server, uses Soapbox UI).

Let's talk about Bluesky.

The good thing with Bluesky.

  1. Since Bluesky is open source*, everyone can see what's happening, live, and on the spot. This also allows other developers to contribute on the Bluesky's code, from the server (ATProto), to the client. If you want to check out behind the scenes of the Bluesky development, visit their GitHub page.

  2. It also does not contain algorithm, unlike Twitter. With Bluesky, you can choose feeds made by everyone. If you want cats only feed, it's there. You want posts, only in one language (e.g. Brazilian Portugese, pt-BR), it's there. The choice is yours.

  3. In case if you want to move your profile, Bluesky can transfer all of your profile preferences to another server.

  4. If you have a website, you can verify yourself with the website domain. (e.g. @theinsidersandrush.bsky.social -> @theinsidersandrush.github.io)

Features that Bluesky improved from Twitter:

  1. Video:

    Bluesky originally added video, up to 60 seconds. Now, Bluesky allows video to be 180 seconds / 3 minutes long.

  2. Verification:

    In 21 April 2025, Bluesky added another form of verification, 'blue tick' verification. Unlike the current state of verification on Twitter, Bluesky uses 'trusted verifier' to verify others. This feature can also be utilised on other non-Bluesky, ATProto-based server, but only available to the said server only. (Example 1: bsky.social account verified by another bsky.social user on deer.social).

Not so good things about Bluesky.

  1. Some users in other countries got banned due to governmental request:

    Freedom of speech was there too on Bluesky, however, some users on Turkey (Republic of Turkiye) got banned by Bluesky's moderation system under request by the Turkish government.

  2. Fierce competition from Twitter:

    Because of Elon Musk's influence over Twitter, many are still in Twitter. In every conversation that I've ever talk, they always bring up a Tweet from Twitter, starting with "X.com" link. Every news, memes, and everything I've heard, always come from Twitter, before Bluesky. Added the factor that there aren't too many other reputable people joining the Bluesky.

Things that I like to improve for Bluesky.

  1. Adding an option to turn off recommendation for politics posts:

    By adding this option, some people can get away not seeing any political posts.

  2. Addition of more languages to Bluesky UI:

    Adding more languages in preparation. The language that I like to see more: Indonesian, Malay, Tamil, Hindi, Thai.

  3. Improving the "Help Center" page:

    Apparently, the help center / support page is on Zendesk's placement URL.Suggesting it to be on https://support.bsky.app/en-us.

Let's talk about Mastodon.

The good thing about Mastodon

  1. Since it's open source*, everyone can see what's inside the code at real time.

  2. No matter what software the server was running (e.g. Misskey, Sharkey (Misskey with Mastodon API), Pixelfed, Peertube, Pleroma), you still can view, comment, and interact posts from one another*. It's called "Fediverse".

  3. Like Bluesky, you can also transfer your account, to another server. (Example: mastodon.social -> mas.to)

Not so good things about Mastodon

  1. Content moderation:

    Every server has different moderation rules. So, if you're being attacked by racists in an another server, it's up to you if you want to block it, or not.

  2. "Dark fediverse" issue:

    If you're in a server that has language barrier, and a lax in moderation, you might find some "Dark fediverse" server, saying slurs, hating on minorities, or using certain media as a point of spreading hate/ideologies. (Example: Far-right anime fans)

  3. incompatibility:

    Some servers might've been modified to suit their use case, causing incompatibility with other clients.

Things that I like to improve for Mastodon

  1. For this time, I don't know what to say about the state of Fediverse at this moment.

End of word.

At last, I've chosen to be always on Bluesky. Although it still lack some major userbase, it's still a calm place to be.

Thank you for reading.

~ Sand, 13 March 2025.
